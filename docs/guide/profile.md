# 个人资料与设置

## 编辑个性签名

1. 登录后，点击导航栏右上角用户名
2. 选择 **编辑资料**
3. 在输入框中修改个性签名（最多 200 字）
4. 点击「保存」

::: tip
个性签名会显示在你的用户主页上，同时也会经过敏感词过滤。
:::

## 用户主页

每个用户都有独立的用户主页，展示：

- **用户名** — 你的昵称
- **个性签名** — 个人简介
- **邮箱绑定状态** — 仅本人可见
- **等级徽章** — 当前等级及颜色
- **Token 到期时间** — 仅本人可见
- **作品列表** — 你发布的所有书籍
- **评价统计** — 你对书籍的评价概况

访问路径：`/user/<用户UUID>`

其他用户访问你的主页时，可以点击 **发私信** 按钮直接与你进行一对一对话。

## 设置页面

登录后访问 **设置** 页面（`/settings`），可以进行以下管理：

### 修改密码

1. 输入当前密码
2. 输入新密码（至少 6 位）
3. 确认新密码
4. 点击「保存」

### 两步验证（2FA）

在设置页面的「两步验证」区域可以管理 2FA：

- **开启 2FA** — 选择「身份验证器应用」或「邮箱验证码」方式，按引导完成设置
- **重新生成备用恢复码** — 旧的恢复码将全部失效，务必保存新的恢复码
- **关闭 2FA** — 需要输入当前密码确认

详细介绍见 [注册与登录 - 两步验证](/guide/auth#两步验证-2fa)。

### 管理 AI 服务

在设置页面可以查看 AI 服务开启状态，首次使用 AI 功能需要阅读并同意 AI 服务条款。

## 绑定邮箱

如果你的账号是在邮箱验证功能上线前注册的，系统会在登录时弹出邮箱绑定提示：

1. 在 **绑定邮箱** 页面输入你的邮箱地址
2. 点击「发送验证码」
3. 输入收到的 6 位验证码（10 分钟内有效）
4. 点击「绑定」

::: tip
邮箱绑定后可用于邮箱验证码登录和密码找回（规划中）。
:::
