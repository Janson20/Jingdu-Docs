# 个人资料设置

## 编辑个性签名

1. 登录后，点击导航栏右上角用户名
2. 选择 **编辑资料**
3. 在输入框中修改个性签名（最多 200 字）
4. 点击「保存」

::: tip
个性签名会显示在你的用户主页上，同时也会经过敏感词过滤。
:::

## 用户主页

每个用户都有独立的用户主页，展示：

- **用户名** — 你的昵称
- **个性签名** — 个人简介
- **邮箱绑定状态** — 仅本人可见
- **等级徽章** — 当前等级及颜色
- **Token 到期时间** — 仅本人可见
- **作品列表** — 你发布的所有书籍
- **评价统计** — 你对书籍的评价概况

访问路径：`/user/<用户UUID>`
