# 注册与登录

## 注册账号

1. 点击导航栏的 **登录/注册** 按钮
2. 切换到 **注册** 选项卡
3. 填写以下信息：
   - **用户名** — 你的昵称
   - **密码** — 至少 6 位
   - **邮箱** — 用于验证和找回密码
   - **邮箱验证码** — 点击「发送验证码」获取（6 位数字，10 分钟内有效）
   - **个性签名**（可选）
4. 完成 Cloudflare 人机验证
5. 点击「注册」按钮

::: tip 注意
- 验证码发送有 60 秒冷却时间
- 验证码最多尝试 5 次
- 用户名和签名会经过敏感词过滤
- 注册功能可能根据站点配置开启或关闭
:::

## 登录

净读支持两种登录方式：

### 密码登录

1. 输入用户名
2. 输入密码
3. 完成人机验证
4. 点击「登录」

### 邮箱验证码登录

1. 切换到「验证码登录」选项
2. 输入邮箱地址
3. 点击「发送验证码」
4. 输入收到的 6 位验证码
5. 点击「登录」

## 登录状态

- 登录成功后会自动跳转回首页
- 登录状态通过 Cookie 和 Token 双重机制维护
- Token 默认有效期为 30 天（网页端）
- 关闭浏览器后再次访问会自动恢复登录状态

## 绑定邮箱（老用户）

如果你的账号是在邮箱验证功能上线前注册的，系统可能会提示你绑定邮箱：

1. 登录后访问 **绑定邮箱** 页面
2. 输入邮箱地址
3. 输入邮箱验证码
4. 点击「绑定」

## 修改密码

1. 登录后访问 **设置** 页面
2. 输入当前密码
3. 输入新密码（至少 6 位）
4. 确认新密码
5. 点击「保存」

## 登出

点击导航栏右上角用户名旁的 **登出** 按钮即可退出登录。
