# 认识净读

**净读** (JingDu) 是一个免费、无广告的综合性阅读社区网站。

## 主要功能

| 功能 | 说明 |
|------|------|
| 书籍阅读 | 浏览和阅读社区书籍，支持分类查找 |
| 写作发布 | 创建和管理自己的书籍作品 |
| 博客系统 | 使用 Markdown 发布博客文章 |
| 论坛交流 | 发帖讨论，楼中楼回复互动 |
| 资源分享 | 分享软件资源，支持版本管理 |
| AI 聊天 | 集成多个 AI 模型，积分制使用 |
| 等级系统 | 10 级等级体系，每日任务获取经验 |
| 聊天室 | 公共实时聊天室 |
| 工具箱 | 可扩展的在线工具集合 |
| B站问号榜 | B 站千粉新号排行榜 |

## 网站导航

登录后，页面顶部导航栏提供以下入口：

- **首页** — 热门推荐、最新书籍、分类浏览、公告、一言
- **图书库** — 所有书籍的分页浏览（每页 32 本）
- **博客** — 社区博客文章列表
- **论坛** — 社区讨论帖子列表
- **资源** — 软件资源分享
- **工具箱** — 在线工具集合
- **AI 聊天** — AI 智能对话（需登录）
- **B站问号榜** — B 站千粉新号排行榜

导航栏右侧功能区：

- **私信** — 消息图标，显示未读消息数（含系统公告未读）
- **等级徽章** — 显示当前等级名称和颜色
- **AI 积分余额** — 显示剩余积分
- **用户名下拉菜单** — 个人主页、编辑资料、设置、等级中心、AI充值、登出

### 其他重要页面入口

- **等级中心** (`/exp`) — 查看经验值和每日任务完成状态
- **AI 充值** (`/ai_recharge`) — 每日签到、兑换码充值、每日任务奖励
- **AI 成本仪表盘** (`/ai_chat/dash`) — 查看 AI 使用成本分析
- **系统公告** (`/message/0`) — 查看管理员发布的站内公告
- **催更历史** (`/cg/历史`) — 查看所有用户的催更记录

## 深色/浅色主题

网站支持深色和浅色主题切换，点击页面右上角的主题切换按钮即可切换。
