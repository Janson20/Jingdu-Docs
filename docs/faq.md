# 常见问题

## 账号相关

### 忘记密码怎么办？

目前暂不支持通过邮箱找回密码。如果你忘记了密码，请联系网站管理员。

### 注册时收不到验证码？

- 检查邮箱地址是否正确
- 检查邮箱的垃圾邮件文件夹
- 确认距离上次发送已超过 60 秒
- 如果仍然无法收到，可能是邮箱服务商的问题

### 用户名有什么限制？

用户名不能包含敏感词，长度有一定限制。建议使用字母、数字和下划线组合。

## 书籍相关

### 如何发布书籍？

详见 [写作发布](/features/writing)。

### 如何上传封面图片？

在新建书籍时，支持上传封面图片。如果不上传，系统会使用默认封面。

### 书籍内容支持什么格式？

书籍章节目前以纯文本形式存储和展示。

## AI 聊天相关

### 积分用完了怎么办？

- 每日签到可以获得 20 积分
- 使用兑换码充值积分

### 支持哪些 AI 模型？

目前支持以下模型：

- DeepSeek V4 Flash（通用对话，模型名：deepseek-v4-flash）
- DeepSeek V4 Pro（深度推理，模型名：deepseek-v4-pro）
- Qwen Plus（通义千问增强版）
- Qwen Flash（通义千问快速版）

> 旧版模型名称（deepseek-chat、deepseek-reasoner）仍可兼容使用。

### AI 聊天记录会保存吗？

聊天记录保存在你的浏览器本地，不会上传到服务器。清除浏览器数据会导致聊天记录丢失。

### 什么是 FIM 补全模式？

FIM（Fill In the Middle）补全模式是 DeepSeek 模型的一项功能，你可以提供内容的开头和结尾，AI 会自动补全中间的部分。适用于续写、代码补全等场景。

### FIM 补全怎么收费？

FIM 补全采用与普通聊天相同的 **基于 token 的阶梯计费**，根据实际输入和输出 token 数量计算积分。开头和结尾内容总长度不超过 1000 字符。仅支持 DeepSeek 系列模型。

### AI 聊天消耗多少积分？

积分消耗取决于实际的 token 用量和所选模型，最低 1 积分。用量越大，后续 token 的兑换率越高。详见 [积分系统说明](/features/ai-chat#积分系统)。

> **大额调用价格说明**：第三阶梯（>10 万 token）的兑换率已调整为与官方单价持平，大额调用无中间差价。平台利润主要来自小额调用的前两阶梯定价及积分向上取整机制。

### AI 聊天支持数学公式吗？

是的！AI 聊天支持渲染 LaTeX 数学公式。在对话中，AI 返回的数学表达式会自动渲染为美观的公式。支持的语法包括：

- **行内公式**：`$...$` 或 `\(...\)`
- **块级公式**：`$$...$$` 或 `\[...\]`

例如：`$\frac{a}{b}$` 会渲染为行内分数，`$$\int_0^1 x^2 dx$$` 会渲染为独立的积分公式。

## 公告相关

### 公告附赠的积分有效期是多久？

管理员在发布公告时可以自定义积分有效期，范围为1-365天，默认30天。过期后用户将无法领取该积分。有效期的起始时间为公告发布时间。

### 如何领取公告附赠的积分？

在公告详情页面，如果公告附带了积分奖励且未过期，会显示"领取"按钮。点击即可领取积分到你的账户。每个公告的积分只能领取一次。

## API 相关

### 如何通过 API 获取自己的用户信息？

调用 `GET /api/user/info` 接口，携带登录时获取的 Token 即可。支持两种方式传递 Token：

- **Authorization 请求头**：`Authorization: Bearer <token>`（推荐）
- **查询参数**：`?token=<token>`

返回信息包括：IP 地址、用户名、UUID、用户简介、等级和 AI 积分余额。
