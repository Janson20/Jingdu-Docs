# 资源分享

分享有用的软件资源，支持版本管理和网盘链接。

## 浏览资源

- 点击导航栏 **资源** 进入资源首页
- 资源列表分页展示，支持搜索功能
- 可以按关键词搜索资源

## 资源详情

点击资源进入详情页，查看：

- **名称** — 资源名称
- **版本** — 当前版本号
- **作者** — 资源上传者
- **简介** — 资源说明
- **官网** — 官方网站链接
- **网盘链接** — 下载地址
- **提取码** — 网盘提取码
- **版本历史** — 所有历史版本记录

## 发布资源

1. 登录后，点击资源页面的 **新建资源** 按钮
2. 填写以下信息：
   - **资源名称**
   - **版本号**
   - **作者**
   - **简介**
   - **官网地址**
   - **网盘链接**
   - **网盘提取码**
3. 点击「发布」

## 管理资源

### 我的资源

登录后，通过资源页面的 **我的资源** 入口查看你发布的所有资源。

### 编辑资源

- 进入资源详情页，点击 **编辑** 按钮（仅所有者可见）
- 修改信息后可以选择 **保存为新版本**，自动生成版本历史
- 也可以直接覆盖当前版本

### 删除资源

- 进入资源详情页，点击 **删除** 按钮（仅所有者可见）

## 搜索资源

在资源首页的搜索框中输入关键词，即可搜索匹配的资源。
