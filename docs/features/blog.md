# 博客系统

使用 Markdown 在净读撰写和发布博客文章，分享你的想法。

## 浏览博客

- 点击导航栏 **博客** 进入博客列表
- 博客列表分页展示（每页 10 篇）
- 博客内容使用 Markdown 渲染，支持代码高亮和目录导航

## 发布博客

1. 登录后，点击博客列表页的 **发布博客** 按钮
2. 输入：
   - **标题** — 博客标题
   - **内容** — 支持 Markdown 格式
3. 点击「发布」

::: info 发布限制
- 每天最多发布 **2 + 等级** 篇博客（如 Lv.3 用户每日上限 5 篇）
- 发布博客可获得 **10 经验值**，每日最多 10 次
:::

## 博客详情

点击博客标题进入详情页，可以查看：

- 博客完整内容（Markdown 渲染为 HTML）
- 评论区
- 作者信息

## 评论博客

1. 在博客详情页底部的评论区输入你的评论
2. 点击「发表评论」

::: tip
- 登录后评论可获得 **3 经验值**，每日最多 10 次
- 评论会显示 IP 归属地
:::

## 管理博客

### 我的博客

登录后，通过博客页面的 **我的博客** 入口查看你发布的所有博客。

### 编辑博客

- 进入博客详情页，点击 **编辑** 按钮（仅作者可见）
- 修改标题和内容后点击「保存」

### 删除博客

- 进入博客详情页，点击 **删除** 按钮（仅作者可见）

### 删除评论

- 博客作者和评论者可以删除评论
