# 书籍阅读

## 浏览书籍

### 首页推荐

首页展示以下内容，无需登录即可浏览：

- **热门推荐** — 4 本热门书籍
- **最新上传** — 8 本最新上传的书籍
- **最新更新** — 4 本最近更新的书籍
- **书籍分类** — 所有可用的书籍分类标签
- **公告** — 网站最新公告
- **一言** — 随机展示一句话

### 图书库

点击导航栏 **图书库** 进入书籍列表页面：

- 书籍以卡片形式展示（封面、标题、作者、评价统计）
- 支持分页浏览（每页 32 本）
- 使用底部页码进行翻页
- 支持按作者、分类和关键词搜索

### 分类搜索

- 在首页点击任意分类标签，可按分类浏览书籍
- 分类标签列出书籍总数
- URL 格式：`/search/<分类名>`

### 搜索书籍

在首页或图书库使用搜索框按关键词搜索书籍，支持匹配书名和作者。

## 书籍详情

点击书籍卡片进入 **书籍详情页**，可以查看：

- **封面图片** — 书籍封面
- **书名** — 书籍标题
- **作者** — 点击作者名可查看该作者的所有作品
- **分类** — 点击分类可浏览同分类书籍
- **简介** — 作者撰写的书籍描述
- **评价统计** — 推荐/不推荐的评价概况（详见 [书籍评价](/features/review)）
- **目录** — 所有章节列表，点击章节标题开始阅读
- **催更按钮** — 点击可提交催更请求（详见 [催更功能](/features/urge)）

## 阅读页面

阅读页面提供：

- **章节内容** — 纯文本展示
- **上一章/下一章** — 章节间快速切换
- **返回目录** — 回到书籍详情页
- **目录跳转** — 点击返回目录后可选择其他章节

::: tip
阅读章节可以获得 **5 经验值**，每日最多 10 次。
:::
