# 书籍阅读

## 浏览书籍

### 首页推荐

首页展示以下内容，无需登录即可浏览：

- **热门推荐** — 4 本热门书籍
- **最新上传** — 8 本最新上传的书籍
- **最新更新** — 4 本最近更新的书籍
- **书籍分类** — 所有可用的书籍分类
- **公告** — 网站最新公告
- **一言** — 随机展示一句话

### 图书库

点击导航栏 **图书库** 进入书籍列表页面：

- 书籍以卡片形式展示（封面、标题、作者、评价统计）
- 支持分页浏览（每页 32 本）
- 使用底部页码进行翻页

### 分类搜索

在首页点击任意分类标签，可按分类浏览书籍。

## 阅读书籍

1. 在图书库或首页点击书籍卡片，进入 **书籍详情页**
2. 在详情页查看书籍信息（封面、标题、作者、分类、描述）
3. 点击章节标题即可开始阅读

## 阅读页面

阅读页面提供：

- **章节内容** — 纯文本展示
- **上一章/下一章** — 章节间快速切换
- **返回目录** — 回到书籍详情页

::: tip
阅读章节可以获得 5 经验值，每日最多 10 次。
:::
