# 书籍阅读

## 浏览书籍

### 首页推荐

首页展示以下内容，无需登录即可浏览：

- **热门推荐** — 4 本热门书籍
- **最新上传** — 8 本最新上传的书籍
- **最新更新** — 4 本最近更新的书籍
- **书籍分类** — 所有可用的书籍分类标签
- **公告** — 网站最新公告
- **一言** — 随机展示一句话

### 图书库

点击导航栏 **图书库** 进入书籍列表页面：

- 书籍以卡片形式展示（封面、标题、作者、评价统计）
- 支持分页浏览（每页 32 本）
- 使用底部页码进行翻页
- 支持按作者、分类和关键词搜索

### 分类搜索

- 在首页点击任意分类标签，可按分类浏览书籍
- 分类标签列出书籍总数
- URL 格式：`/search/<分类名>`

### 搜索书籍

在首页或图书库使用搜索框按关键词搜索书籍，支持匹配书名和作者。

## 书籍详情

点击书籍卡片进入 **书籍详情页**，可以查看：

- **封面图片** — 书籍封面
- **书名** — 书籍标题
- **作者** — 点击作者名可查看该作者的所有作品
- **分类** — 点击分类可浏览同分类书籍
- **简介** — 作者撰写的书籍描述
- **评价统计** — 推荐/不推荐的评价概况（详见 [书籍评价](/features/review)）
- **目录** — 所有章节列表，点击章节标题开始阅读
- **催更按钮** — 点击可提交催更请求（详见 [催更功能](/features/urge)）
- **AI 整本概括** — 一键将全书所有章节发送至 AI 聊天，请求对整本书进行概括分析

### AI 整本概括

在书籍详情页的操作按钮区域，有一个 **🤖 整本概括** 按钮：

1. 点击按钮，弹出确认窗口
2. 窗口会显示书籍名称、章节总数，并提醒"概括可能消耗较多积分"
3. 确认后，系统自动拼接全部章节内容为文本文件
4. 跳转到 AI 聊天页面，AI 自动对整本书进行全面概括

::: warning
整本概括需要处理全书所有章节，消耗的积分较多且耗时更长。建议在积分充裕时使用。
:::

## 阅读页面

阅读页面提供：

- **章节内容** — 纯文本展示
- **上一章/下一章** — 章节间快速切换
- **返回目录** — 回到书籍详情页
- **目录跳转** — 点击返回目录后可选择其他章节
- **AI 概括本章** — 页面右侧浮动按钮，一键将当前章节发送至 AI 聊天并请求概括

### AI 概括本章

在阅读页面右侧，有一个紫色的 **🤖 概括** 浮动按钮：

1. 点击按钮，系统自动将当前章节内容拼接为文本文件
2. 自动跳转到 AI 聊天页面，附上章节文件
3. AI 自动发送概括指令，从核心情节、人物关系变化、关键事件与转折点三个方面概括本章

::: tip
使用 AI 概括功能需要登录并开通 AI 服务。概括会消耗 AI 积分，详见 [AI 聊天计费规则](/features/ai-chat#积分系统)。
:::

### 阅读设置

在阅读页面，点击右侧的 **⚙ 设置按钮** 或按键盘 `S` 键打开阅读设置面板，可以自定义以下选项：

| 设置项 | 选项 | 说明 |
|--------|------|------|
| 字号 | 小 / 中 / 大 / 超大 | 调整正文字体大小 |
| 行间距 | 紧凑(1.5) / 标准(1.8) / 宽松(2.0) / 极宽(2.5) | 调整正文行间距 |
| 字体 | 默认 / 宋体 / 黑体 | 切换正文字体 |
| 背景色 | 白色 / 护眼绿 / 羊皮纸 / 夜间 | 切换阅读背景主题 |

**快捷键列表：**
- `←` 上一章 · `→` 下一章
- `S` 打开/关闭设置面板 · `T` 回顶部
- `Esc` 关闭设置面板

**移动端手势：** 左滑翻到下一章，右滑翻到上一章。

设置会先保存到浏览器本地，登录后自动同步到云端，跨设备保持一致。

::: tip
阅读章节可以获得 **5 经验值**，每日最多 10 次。
:::
