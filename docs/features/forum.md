# 论坛交流

在论坛发帖和讨论，与社区成员互动。

## 浏览帖子

- 点击导航栏 **论坛** 进入帖子列表
- 帖子以随机推荐排序展示，增加内容曝光
- 分页浏览帖子列表
- 每个帖子显示浏览量和回复数

## 发布帖子

1. 登录后，点击论坛页面的 **发布帖子** 按钮
2. 输入：
   - **标题** — 帖子标题
   - **内容** — 帖子正文
3. 点击「发布」

::: info 发布限制
- 每天最多发布 **10 + 等级** 个帖子（如 Lv.3 用户每日上限 13 个）
- 发布帖子可获得 **10 经验值**，每日最多 10 次
:::

## 帖子详情

点击帖子标题进入详情页，可以看到：

- 帖子完整内容
- 所有回复（支持楼中楼结构）
- 表情自动转换（如 `:)` 会自动变成对应表情）

## 回复帖子

1. 在帖子详情页输入回复内容
2. 可以引用其他楼层进行回复（楼中楼）
3. 点击「发表回复」

::: tip
- 登录后评论可获得 **3 经验值**，每日最多 10 次
- 回复会显示 IP 归属地
- 支持 `:)` 等表情代码自动转换为 emoji
:::

## 管理帖子

### 我的帖子

登录后，通过论坛页面的 **我的帖子** 入口查看你发布的所有帖子。

### 编辑帖子

- 进入帖子详情页，点击 **编辑** 按钮（仅作者可见）
- 修改标题和内容后点击「保存」

### 删除帖子

- 进入帖子详情页，点击 **删除** 按钮（仅作者可见）

### 删除评论

- 帖子作者和评论者可以删除评论
