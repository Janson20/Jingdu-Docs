# 论坛交流

在论坛发帖和讨论，与社区成员互动。

## 浏览帖子

- 点击导航栏 **论坛** 进入帖子列表
- 帖子以随机推荐排序展示，增加内容曝光
- 分页浏览帖子列表
- 每个帖子显示浏览量和回复数

## 发布帖子

1. 登录后，点击论坛页面的 **发布帖子** 按钮
2. 输入：
   - **标题** — 帖子标题
   - **内容** — 帖子正文
3. 点击「发布」

::: info 发布限制
- 每天最多发布 **10 + 等级** 个帖子（如 Lv.3 用户每日上限 13 个）
- 发布帖子可获得 **10 经验值**，每日最多 10 次
:::

## 帖子详情

点击帖子标题进入详情页，可以看到：

- 帖子完整内容
- 所有回复（支持楼中楼结构）
- 表情自动转换（如 `:)` 会自动变成对应表情）

## 回复帖子

1. 在帖子详情页输入回复内容
2. 可以引用其他楼层进行回复（楼中楼）
3. 点击「发表回复」

::: tip
- 登录后评论可获得 **3 经验值**，每日最多 10 次
- 回复会显示 IP 归属地
- 支持 `:)` 等表情代码自动转换为 emoji
:::

## 管理帖子

### 我的帖子

登录后，通过论坛页面的 **我的帖子** 入口查看你发布的所有帖子。

### 编辑帖子

- 进入帖子详情页，点击 **编辑** 按钮（仅作者可见）
- 修改标题和内容后点击「保存」

### 删除帖子

- 进入帖子详情页，点击 **删除** 按钮（仅作者可见）

### 删除评论

- 帖子作者和评论者可以删除评论

## 帖子互动

在帖子详情页，帖子内容下方有互动按钮栏，你可以：

### 点赞

- 点击 ❤️ **点赞** 按钮为帖子点赞，再次点击取消
- 帖子作者每被点赞可获得 **1 经验值**（每日上限 50 次）
- 登录后在 **我的点赞** 页面可查看你点赞过的所有帖子

### 收藏

- 点击 ⭐ **收藏** 按钮收藏帖子，再次点击取消收藏
- 帖子作者每被收藏可获得 **2 经验值**（每日上限 30 次）
- 登录后在 **我的收藏** 页面可查看你收藏过的所有帖子

### 转发给好友

- 点击 ↗️ **转发** 按钮，选择私信好友即可将帖子摘要转发给对方
- 转发内容包括帖子标题、作者、前 200 字摘要和帖子链接
- 不能转发给已拉黑的用户或自己
- 帖子作者每被转发可获得 **3 经验值**（每日上限 20 次）

### 查看我的互动

在论坛首页或「我的帖子」页面，通过 tab 标签可切换查看：

- **我的帖子** — 你发布的所有帖子
- **我的点赞** — 你点赞过的帖子
- **我的收藏** — 你收藏过的帖子
