# 书籍评价

净读采用类似 Steam 的评价系统，为每本书提供直观的评价概览。

## 评价方式

每本好书详情页都有评价面板，你可以选择：

- **推荐** — 觉得这本书不错
- **不推荐** — 觉得这本书不够好

你还可以填写文字评价来详细说明你的看法。

## 评价规则

- 每个用户对每本书只能提交 **一条** 评价
- 可以随时 **修改** 你的评价（重新选择推荐/不推荐 + 修改文字）
- 登录后才能提交评价

## 评价统计

书籍详情页会显示整体评价统计：

| 评价区间 | 说明 |
|---------|------|
| 好评如潮 | 推荐比例 >= 95% |
| 多半好评 | 推荐比例 >= 80% |
| 褒贬不一 | 推荐比例 >= 50% |
| 多半差评 | 推荐比例 >= 20% |
| 差评如潮 | 推荐比例 < 20% |

## 获取经验

- 评价书籍（文字超过 10 字）可获得 **5 经验值**
- 每日最多获取 **5 次** 评价经验
