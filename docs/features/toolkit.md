# 工具箱

净读提供可扩展的在线工具集合平台。

## 浏览工具

- 点击导航栏 **工具箱** 进入工具列表页（`/toolkit`）
- 每个工具以卡片形式展示，包含工具名称、图标和简介
- 点击工具卡片可查看详细说明

## 使用工具

1. 在工具箱列表页点击 **使用** 按钮
2. 进入工具页面进行操作
3. 每个工具都有独立的操作界面

## 工具框架

工具箱采用 **可扩展框架** 设计，支持动态添加新工具：

- 每个工具通过配置文件定义名称、描述、图标和入口 URL
- 部分工具以 **iframe 嵌入** 方式运行外部页面
- 支持独立工具说明页面（`/tool-info/<id>`）

## 已有工具

工具箱目前提供以下实用工具（具体列表以网站实际展示为准）：

- 各类文本处理、格式转换等在线小工具
