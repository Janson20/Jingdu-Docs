# 写作发布

成为作者，在净读发布你的原创作品。

## 作者写作面板

1. 登录后，在导航栏找到 **写作** 入口
2. 进入 **写作面板**，展示你所有的作品
3. 每个作品可以查看章节列表和添加新章节

## 新建书籍

1. 点击写作面板中的 **新建书籍** 按钮
2. 填写书籍信息：
   - **书名** — 书籍标题
   - **简介** — 书籍描述
   - **分类** — 选择一个书籍分类
   - **封面** — 上传封面图片（可选）
3. 点击「创建」按钮

::: tip
- 书名和简介会经过敏感词过滤
- 封面图片为可选项，不上传会使用默认封面
:::

## 添加章节

1. 在写作面板中找到目标书籍
2. 点击 **添加章节**
3. 输入：
   - **章节标题**
   - **章节内容**
4. 点击「保存」

章节保存后会自动追加到书籍末尾。

## 管理作品

- 在写作面板可以查看所有已发布的书籍
- 每本书显示章节列表
- 可以随时添加新章节

## 用户主页

你发布的所有书籍会在 **用户主页** 中展示，其他用户可以通过 `/user/<你的UUID>` 访问你的主页查看你的作品。

## 获取经验

- 每发布一个内容（章节等）可获得 **10 经验值**
- 每日最多获取 **10 次** 发布经验
