# AI 聊天

净读集成了 DeepSeek 和 Qwen 大语言模型，提供智能 AI 对话服务。

## 首次使用

### 开通服务

1. 登录后，点击导航栏的 **AI 聊天** 入口
2. 首次使用需要阅读并同意 **AI 服务条款**
3. 点击 **开启服务** 按钮

## 积分系统

AI 聊天使用 **基于 token 的阶梯计费** 制，1 积分 = 0.01 元人民币。

### 计费规则

每次请求根据实际的 **输入 token 数** 和 **输出 token 数** 计算积分，向上取整（最低 1 积分）。

采用 **超额累进阶梯** 计费，用量越高，后续 token 的兑换率越高（单价越低）：

| 阶梯 | token 用量范围 |
|------|----------------|
| 第一阶梯 | 0 ~ 10,000 |
| 第二阶梯 | 10,001 ~ 100,000 |
| 第三阶梯 | 100,001 及以上 |

### 各模型阶梯兑换率（token/积分）

**DeepSeek V4 Pro（深度推理）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 800 | 400 |
| 第二阶梯（1万~10万） | 800 | 400 |
| 第三阶梯（10万以上） | 833 | 416 |

**DeepSeek V4 Flash（通用对话）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 5,000 | 2,500 |
| 第二阶梯（1万~10万） | 8,000 | 4,000 |
| 第三阶梯（10万以上） | 10,000 | 5,000 |

**Qwen Plus（通义千问增强版）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 6,000 | 2,500 |
| 第二阶梯（1万~10万） | 10,000 | 4,500 |
| 第三阶梯（10万以上） | 12,500 | 5,000 |

**Qwen Flash（通义千问快速版）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 6,000 | 2,500 |
| 第二阶梯（1万~10万） | 12,000 | 4,500 |
| 第三阶梯（10万以上） | 20,000 | 5,000 |

> 积分在请求完成后根据实际 token 用量扣除（后置计费）。

### 获取积分

- **每日签到** — 每次 +5 积分
- **每日任务** — 完成指定任务手动领取奖励
- **兑换码** — 使用兑换码充值积分

### 签到

1. 登录后访问 AI 充值页面
2. 点击 **每日签到** 按钮
3. 每日仅可签到一次，签到后获得 5 积分

### 每日任务

完成任务可获得额外积分奖励，每日凌晨刷新，奖励需手动领取：

| 任务 | 奖励 | 说明 |
|------|------|------|
| 和模型对话一次 | +3 积分 | 向任意模型发送一条消息 |
| 上传文件与模型对话一次 | +5 积分 | 上传文件后向模型提问 |
| 体验高级模型一次 | +5 积分 | 使用 DeepSeek V4 Pro 完成一次对话 |

任务进度可在 AI 充值页面查看，完成任务后点击 **领取奖励** 按钮即可获得对应积分。

### 兑换码充值

1. 登录后访问 AI 充值页面
2. 在兑换码输入框中输入兑换码
3. 点击 **兑换**

## 使用 AI 聊天

1. 进入 **AI 聊天** 页面
2. 选择 AI 模型（下拉菜单）
3. 在输入框中输入你的问题
4. 点击发送按钮或按回车

### 书籍搜索增强

AI 聊天支持 **书籍搜索增强** 功能：

- 开启后，AI 会判断是否需要搜索书库信息来辅助回答
- 如果你询问关于站内书籍的内容，AI 会自动检索相关信息

### AI 人设

AI 聊天支持 **人设系统**，可以让 AI 扮演特定角色，提供更专业的回答。

#### 使用方式

1. 在左侧边栏的 **使用人设** 下拉菜单中选择角色
2. 选择后，聊天区域会显示该人设的推荐问题
3. 点击推荐问题即可自动填入输入框
4. 选择 **默认（无人设）** 可恢复通用 AI 助手模式
5. 人设切换后，之前对话的上下文仍然保留

#### 可选人设

| 人设 | 说明 |
|------|------|
| 全能写作助手 | 精通商务邮件、学术论文、营销文案、创意故事等文体 |
| 编程随行导师 | 精通 Python、JavaScript、Java、Web 开发等，帮助调试和学习 |
| 高效学习教练 | 精通费曼学习法、间隔重复等学习策略，制定学习计划 |
| 心灵树洞伙伴 | 基于积极心理学提供情绪支持和倾听 |
| 私人健身顾问 | 根据身体状况和目标设计个性化训练计划 |
| 法律百事通 | 解释法律概念，指导法律文书撰写 |
| 营养饮食规划师 | 定制个性化饮食计划与食谱 |
| 职场领航者 | 简历优化、面试模拟、职业路径规划 |
| 旅行智囊团 | 推荐目的地、规划行程、提供旅行建议 |
| 生活妙招管家 | 家居收纳、时间管理、理财小技巧 |

> 人设系统通过系统提示词（System Prompt）生效，每次对话会将对应人设的提示词发送给 AI 模型。

### FIM 补全模式

使用 DeepSeek 模型时，可以开启 **FIM 补全模式**（Fill In the Middle）：

- 提供 **开头** 和 **结尾**（可选），AI 补全中间的内容
- 适用于内容续写、代码补全等场景
- 开头和结尾总长度不超过 1000 字符
- 计费方式与普通聊天相同，基于实际 token 用量阶梯计费

::: tip
FIM 补全仅支持 DeepSeek 系列模型（deepseek-v4-flash、deepseek-v4-pro）。旧版模型名称（deepseek-chat、deepseek-reasoner）仍可兼容使用。
:::

### 对话轮数限制

- 每次对话最多支持 **50 轮**（一问一答为 1 轮）
- 达到上限后需要清空对话才能继续
- 清空对话后历史记录不会保留

### 聊天历史

- 聊天记录保存在浏览器本地
- 你可以在当前会话中查看上下文
- 切换模型会开始新的对话

### 查询余额

点击聊天页面的余额显示区域，或访问充值页面查看当前积分余额。

### AI 成本仪表盘

访问 `/ai_chat/dash` 查看你的 AI 使用成本分析：

- **总览卡片** — 本月消耗积分、折合人民币、官方同量价格对比、节省金额、请求总数、第三阶梯请求占比
- **请求详解** — 按时间倒序的请求账单，点击任意请求可展开**计费拆解卡片**，查看阶梯计费的完整分解过程
- **阶梯利用率矩阵** — 各模型在不同阶梯的分布热力图，直观了解用量结构
- **自定义对比计算器** — 输入任意 token 数和模型，即时查看阶梯明细和官方价格对比

仪表盘数据来源于每次请求的实际 token 用量，完全透明可审计。

## 文件上传解析

AI 聊天支持上传文件并自动解析内容，让 AI 帮你分析文档：

### 支持的文件格式

支持 **40+ 种文件格式**，包括：

| 类型 | 格式 |
|------|------|
| PDF | `.pdf` |
| Word | `.docx`, `.doc` |
| Excel | `.xlsx`, `.xls`, `.csv` |
| PowerPoint | `.pptx` |
| 文本 | `.txt`, `.md`, `.json`, `.xml`, `.yaml`, `.ini`, `.log` |
| 代码 | `.py`, `.js`, `.ts`, `.html`, `.css`, `.java`, `.c`, `.cpp`, `.go`, `.rs`, `.php`, `.rb`, `.swift`, `.kt`, `.sh`, `.sql` |
| 其他 | `.rtf`, `.epub`, `.html` 等 |

### 如何使用

1. 在 AI 聊天页面上传文件（文件大小限制以界面提示为准）
2. 系统自动解析文件内容为文本
3. 上传成功后，在对话框中向 AI 提问关于文件内容的问题
4. AI 会基于文件内容进行回答

::: tip
- 上传文件与模型对话一次可获得 **每日任务奖励**（+5 积分）
- 解析内容过长时（超过 1000000 字符）会自动截断
:::

## 思考过程展示

使用 DeepSeek V4 Pro（deepseek-v4-pro）等支持推理的模型时，AI 的 **思考过程**（`reasoning_content`）会在聊天界面中展示：

- 思考内容以可折叠区域显示
- 支持 Markdown 渲染，方便阅读推理逻辑
- 帮助你了解 AI 是如何得出结论的
- 思考过程不计入积分消耗（仅计算最终输出 token）

## LaTeX 数学公式

AI 聊天支持渲染 LaTeX 数学公式。对话中的数学表达式会自动渲染为美观的公式：

- **行内公式**：`$...$` 或 `\(...\)`
- **块级公式**：`$$...$$` 或 `\[...\]`

例如：
- `$\frac{a}{b}$` 渲染为行内分数
- `$$\int_0^1 x^2 dx$$` 渲染为独立的积分公式

::: tip
- AI 聊天页面签到可获得 **5 经验值**（每日 1 次）
- 每日签到赠送 **5 积分**，完成每日任务可额外获取积分
- 聊天消息会经过敏感词过滤
:::
