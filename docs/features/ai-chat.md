# AI 聊天

净读集成了 DeepSeek 和 Qwen 大语言模型，提供智能 AI 对话服务。

## 首次使用

### 开通服务

1. 登录后，点击导航栏的 **AI 聊天** 入口
2. 首次使用需要阅读并同意 **AI 服务条款**
3. 点击 **开启服务** 按钮

## 积分系统

AI 聊天使用 **基于 token 的阶梯计费** 制，1 积分 = 0.01 元人民币。

### 计费规则

每次请求根据实际的 **输入 token 数** 和 **输出 token 数** 计算积分，向上取整（最低 1 积分）。

采用 **超额累进阶梯** 计费，用量越高，后续 token 的兑换率越高（单价越低）：

| 阶梯 | token 用量范围 |
|------|----------------|
| 第一阶梯 | 0 ~ 10,000 |
| 第二阶梯 | 10,001 ~ 100,000 |
| 第三阶梯 | 100,001 及以上 |

### 各模型阶梯兑换率（token/积分）

**DeepSeek V4 Pro（深度推理）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 200 | 100 |
| 第二阶梯（1万~10万） | 400 | 200 |
| 第三阶梯（10万以上） | 600 | 300 |

**DeepSeek V4 Flash（通用对话）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 4,000 | 2,000 |
| 第二阶梯（1万~10万） | 6,000 | 3,000 |
| 第三阶梯（10万以上） | 8,000 | 4,000 |

**Qwen Plus（通义千问增强版）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 4,000 | 2,000 |
| 第二阶梯（1万~10万） | 7,000 | 3,000 |
| 第三阶梯（10万以上） | 10,000 | 4,000 |

**Qwen Flash（通义千问快速版）：**

| 阶梯 | 输入 | 输出 |
|------|------|------|
| 第一阶梯（0~1万） | 8,000 | 1,000 |
| 第二阶梯（1万~10万） | 14,000 | 1,800 |
| 第三阶梯（10万以上） | 20,000 | 2,700 |

> 积分在请求完成后根据实际 token 用量扣除（后置计费）。

### 获取积分

- **每日签到** — 每次 +20 积分
- **兑换码** — 使用兑换码充值积分

### 签到

1. 登录后访问 AI 充值页面
2. 点击 **每日签到** 按钮
3. 每日仅可签到一次，签到后获得 20 积分

### 兑换码充值

1. 登录后访问 AI 充值页面
2. 在兑换码输入框中输入兑换码
3. 点击 **兑换**

## 使用 AI 聊天

1. 进入 **AI 聊天** 页面
2. 选择 AI 模型（下拉菜单）
3. 在输入框中输入你的问题
4. 点击发送按钮或按回车

### 书籍搜索增强

AI 聊天支持 **书籍搜索增强** 功能：

- 开启后，AI 会判断是否需要搜索书库信息来辅助回答
- 如果你询问关于站内书籍的内容，AI 会自动检索相关信息

### FIM 补全模式

使用 DeepSeek 模型时，可以开启 **FIM 补全模式**（Fill In the Middle）：

- 提供 **开头** 和 **结尾**（可选），AI 补全中间的内容
- 适用于内容续写、代码补全等场景
- 开头和结尾总长度不超过 1000 字符
- 计费方式与普通聊天相同，基于实际 token 用量阶梯计费

::: tip
FIM 补全仅支持 DeepSeek 系列模型（deepseek-v4-flash、deepseek-v4-pro）。旧版模型名称（deepseek-chat、deepseek-reasoner）仍可兼容使用。
:::

### 聊天历史

- 聊天记录保存在浏览器本地
- 你可以在当前会话中查看上下文
- 切换模型会开始新的对话

### 查询余额

点击聊天页面的余额显示区域，或访问充值页面查看当前积分余额。

::: tip
- AI 聊天页面签到可获得 **5 经验值**（每日 1 次）
- 聊天消息会经过敏感词过滤
:::
