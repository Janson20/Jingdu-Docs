# 聊天室

净读提供公共聊天室，供所有用户实时交流。

## 进入聊天室

点击导航栏中的 **聊天室** 入口（`/chat`）。

## 使用聊天室

- 在输入框中输入消息，点击发送按钮或按 **Enter** 键发送
- 消息会实时显示在聊天窗口中，支持自动刷新
- 聊天室最多保留最近 **1000 条**消息
- 所有消息对所有用户可见，是公共交流空间

## 消息格式

- 消息为纯文本格式
- 所有消息在发送前会经过**敏感词过滤**

## 获取经验

- 访问聊天室页面可获得 **5 经验值**（计入「访问导航栏页面」任务）

::: info
- 需要登录才能发送消息
- 未登录用户也可以查看聊天记录
- 聊天室历史消息不支持删除（自动滚动清理）
- 刷屏、广告等行为违反 [社区规范](https://jingdu.qzz.io/community)
:::
